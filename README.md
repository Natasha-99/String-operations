## Операции cо строками

Учебный проект для отработки работы со строками на **JavaScript**. В проекте реализовано 12 типовых задач: подсчёт вхождений буквы, проверка, является ли слово числом, удаление пробелов, формирование инициалов, манипуляции с символами и другие.

**Стек:**
* **HTML** - разметка интерфейса;
* **CSS** - стили и оформление
* **JavaScript** - логика и вычисления.

Введите данные в поля ввода, нажмите «Выполнить» и для каждой задачи получите результат сразу под кнопкой.
